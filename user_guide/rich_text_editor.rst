=================
富文本编辑器
=================

Admin文本编辑器
==================

DeerU的后台管理系统使用了 **Froala** 编辑器，这是一个收费的编辑器。

你可以在网上搜寻Django的富文本编辑器，
然后在 ``deeru/settings_local.py`` 中添加、修改 :ref:`富文本编辑器的配置 <DEERU-RICH-EDITOR>` 进行替换。



评论文本编辑器
==================
文章页评论的文本编辑器为主题自定义的，DeerU并没有为这个编辑器做过多规定。
